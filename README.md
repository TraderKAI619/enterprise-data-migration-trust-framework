## Business Scenario

A global pharmaceutical company currently manages patent and trademark information across multiple legacy systems.

Challenges include:

- Multiple status definitions
- Multiple country code standards
- Missing inventor information
- Duplicate records
- Inconsistent business rules

The goal is to standardize and migrate data into a trusted reporting environment.

---

## Project Goals

- Understand client requirements
- Profile and assess source data quality
- Create value mapping specifications
- Implement transformation logic
- Validate migration accuracy
- Deliver trusted reporting outputs
